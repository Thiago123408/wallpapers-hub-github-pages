Sobre o projeto

Este site é hospedado no GitHub Pages e permite que qualquer pessoa contribua com papéis de parede através de Issues. Um GitHub Action lê as imagens postadas nos issues, baixa para a pasta /wallpapers/ do repositório e atualiza o wallpapers/index.json, que é lido por este site.

Como contribuir

  1. Clique em Enviar imagem (abre um issue com modelo).
  2. Anexe suas imagens (PNG/JPG/WEBP/GIF) e preencha as informações.
  3. Envie o issue. O robô (Actions) baixa os arquivos, cria o commit e atualiza a galeria.

Todos os arquivos ficam públicos para qualquer pessoa baixar. Utilize apenas imagens que você tem permissão para distribuir.
